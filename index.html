Misidentified is a block game reflecting the lack of inclusiveness in datasets of AI systems.
Select blocks of more than one pixel and the same colour




Tap on one of the four different modes each representing a stage of data sets
evolution for facial recognition of individuals with different skin tones.

1 2 3 4

Present datasets

The first stage is what most applications have, which is a lot of people with lighter skin tones than people with darker skin tones. The game shows this by having fewer darker tones than lighter tones and you'll be left with the most reflected data in existing datasets when you finish playing-lighter skin tones.

Alternative of present datasets

The second stage poses the question: what if? If the tables were turned and current datsets represented more darker skin tones than lighter skin tones. The games shows a lot more darker tones than lighters and it will be easier for the user to select darker tones as it will be easier to distiguish than the lighter tones.

This is not objective to one skin tone

The third level shows what if we both had a set of data equivalent for both skin tones, so it was still challenging to make a difference between tones.

Optimal dataset

the final level shows the optimal set of data that is so inclusive and the program will make a clear difference, which is what we should promote.
